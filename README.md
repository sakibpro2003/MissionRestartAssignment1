# SwiftCart E-Commerce

A responsive e-commerce website built with HTML, Tailwind CSS, DaisyUI, and Vanilla JavaScript using Fake Store API.

## Live & Repository

- **Live Link:** `YOUR_DEPLOYED_URL_HERE`
- **GitHub Repository:** `YOUR_PUBLIC_REPO_URL_HERE`

## Features Implemented

- Responsive navbar with logo, menu, and cart count badge
- Hero/banner section
- Why Choose Us section
- Trending products section (API)
- All products page
- Dynamic category filtering
- Product details modal (single product API)
- Add to cart count using localStorage
- About page and Contact page
- Styled newsletter + footer

## API Endpoints Used

- All Products: `https://fakestoreapi.com/products`
- All Categories: `https://fakestoreapi.com/products/categories`
- Products by Category: `https://fakestoreapi.com/products/category/${category}`
- Single Product Detail: `https://fakestoreapi.com/products/${id}`

## Technology Stack

- HTML
- Tailwind CSS
- DaisyUI
- Vanilla JavaScript
